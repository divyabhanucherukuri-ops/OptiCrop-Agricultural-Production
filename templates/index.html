Powered by Machine Learning

OptiCrop uses a tuned Random Forest Classifier achieving 99% accuracy on the crop recommendation dataset. The model analyzes 7 key parameters and compares them against 2,200+ data samples spanning 22 different crop types.

  • 5 algorithms compared (KNN, Logistic Regression, Decision Tree, Random Forest, Gradient Boosting)
  • Hyperparameter tuning via GridSearchCV for optimal performance
  • Feature scaling with StandardScaler for accurate predictions
  • Cross-validation to ensure model generalizability
